## 🎯 Funcionalidades Principais

### 🛠️ CRUD Completo
- **Cadastro de Eventos**:
  - Título do evento
  - Data
  - Capacidade
  - Localização
  - Preço do ingresso
  - Descrição completa

- **Recursos Avançados**:
  - 🔍 Busca textual
  - 📅 Filtro por período de datas
  - 💰 Filtro por faixa de preço
